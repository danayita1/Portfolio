# University Course Catalog Database

## Project Goal
This project demonstrates my ability to design and manage a relational database using SQL and MySQL. It showcases database creation, table design, data insertion, and queries to extract meaningful insights.

## Skills Used
- SQL (Data Definition & Manipulation)
- MySQL / MySQL Workbench
- Data Analysis & Reporting
- Data Management Concepts

## Project Description
I designed a **university course catalog database** with three tables: `Students`, `Courses`, and `Enrollments`. I populated the tables with realistic sample data and wrote queries to:

1. List all students
2. Show enrollments per student with grades
3. Calculate average grade per course
4. Identify students with an A grade
5. Count the number of students per course

This project demonstrates my skills in database design, data querying, and analysis.

## Files
- `create_tables.sql` – Script to create tables and insert sample data.
- `portfolio_queries.sql` – Queries used for portfolio demonstrations.
- `screenshots/` – Screenshots of query outputs from MySQL Workbench.

## How to Run
1. Open **MySQL Workbench**.  
2. Run `create_tables.sql` to create the database, tables, and insert sample data.  
3. Run `portfolio_queries.sql` to see all example queries and their results.
