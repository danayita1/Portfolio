# SQL Database Project

**Goal:** Build a university course catalog database and analyze student enrollment and grades.

**Skills used:** SQL, Java (JDBC optional), Data Management, Data Analysis

**Description:**  
This project demonstrates my ability to design relational databases, populate tables with realistic data, and write queries to extract meaningful insights.  

**Database Structure:**  
- Students: student_id, first_name, last_name, major, year  
- Courses: course_id, course_name, course_code, credits  
- Enrollments: enrollment_id, student_id, course_id, grade  

**Key Queries / Insights:**  
1. List all students in a course  
2. Calculate average grade per course  
3. Identify students at risk (grade < C)  
4. Count students per major  

**Status:** Work in progress
